# Verification Bot

![Discord](https://img.shields.io/badge/Discord-Bot-blue)

**Verification Bot** is a Discord bot for automatic user verification using image CAPTCHA.  
It manages **Verified / Unverified** roles, allows log channel setup, and restricts commands to the server owner.

- Modular command system (`commands/`)  
- Multi-server support
